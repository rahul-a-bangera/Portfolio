?? Portfolio API Documentation

RESTful API Endpoints for Portfolio Data

?? About This API

This API provides access to portfolio data including contact information, resume details, and blog posts.

Base URL: https://gentle-moss-0d321ce00.2.azurestaticapps.net/api

Authentication: None required (public API)

Response Format: JSON

CORS: Enabled for all origins

?? Main Website

For the full portfolio website, visit: https://rahul-a.in

This page serves only as API documentation. The complete portfolio frontend is hosted on GitHub Pages.

?? Available Endpoints

GET Contact Information
/api/contact
Returns contact information including email, phone, and social media links.
GET Resume Data
/api/resume
Returns complete resume data including personal info, summary, skills, experience, education, certifications, and projects.
GET Blog Posts
/api/blog
Returns a list of all blog posts with id, slug, title, description, author, publish date, tags, and read time.
GET Blog Post by Slug
/api/blog/{slug}
Returns a single blog post by its slug identifier. Replace {slug} with the actual blog post slug.
