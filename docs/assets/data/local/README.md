# Local Development Data

This folder contains your personal data for local development.

**IMPORTANT: These files are NOT tracked in Git for privacy.**

## Setup

1. Copy template files to this folder:
   ```bash
   cp templates/*.json .
   ```

2. Edit the JSON files with your personal information.

3. Run the app locally:
   ```bash
   npm start
   ```

## Files

- `profile.json` - Your name and specialist content
- `contact.json` - Your contact information  
- `resume.json` - Your complete resume data

## Privacy

? These files are in `.gitignore`  
? Will NOT be committed to Git  
? Safe for personal information  

**Never remove these files from .gitignore!**
